# 🧰 ابزارهای سه‌گانه پردازش متن

یک ابزار سبک و قدرتمند برای پردازش لیست‌های متنی، مناسب برای مدیران شبکه، برنامه‌نویسان و هر کسی که با داده‌های متنی سروکار دارد.

## ✨ ویژگی‌ها

### ۱. افزودن متن و عدد به لیست
- اضافه کردن پیشوند (متن ثابت) به هر خط
- افزودن شماره ترتیبی (با شروع دلخواه)
- مناسب برای تولید لیست‌های شماره‌دار

### ۲. پاک کردن از کاراکتر به بعد
- حذف هر چیزی بعد از یک کاراکتر مشخص (مثل `:` یا `#`)
- مناسب برای پاک کردن توضیحات اضافی از لیست‌ها

### ۳. گروه‌بندی پورت‌ها بر اساس IP
- تشخیص آی‌پی‌های تکراری
- نمایش پورت‌های تکراری با برچسب 🔴
- استخراج خروجی خطی (ip:port) بدون تکرار

## 🚀 نحوه استفاده

1. در هر تب، داده‌های خود را وارد کنید
2. دکمه مربوطه را بزنید
3. نتیجه را کپی کنید

## 💾 ذخیره‌سازی خودکار

همه ورودی‌ها به‌طور خودکار در مرورگر شما ذخیره می‌شوند (با localStorage) — پس از رفرش صفحه، داده‌ها از بین نمی‌روند.

## 🛠 تکنولوژی‌ها

- HTML5
- CSS3 (ریسپانسیو)
- JavaScript خالص (بدون وابستگی)

## 📱 پشتیبانی از موبایل

طراحی کاملاً واکنش‌گرا (Responsive) برای استفاده در همه دستگاه‌ها.
